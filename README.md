# Python Console Quiz Game

Python으로 만든 콘솔 기반 퀴즈 게임입니다. 터미널에서 문제를 풀고, 새 문제를 추가하고, 최고 점수를 저장할 수 있는 간단한 학습용 프로그램입니다.

# 1. 프로젝트 개요
개발 언어: Python 3

주요 기능: 퀴즈 풀기, 퀴즈 등록, 목록 조회, 최고 점수 기록, JSON 파일 저장

특징: 클래스 분리, 숫자 입력 검증, 파일 저장 중 예외 처리

# 2. 퀴즈 주제 선정 이유
주제: 리눅스 기본 명령어, Docker, Git

기본 문제는 학습 초반에 자주 접하는 명령어 위주로 구성했습니다. 터미널 사용, 버전 관리, 컨테이너 기초 내용을 가볍게 복습하는 목적에 맞췄습니다.

# 3. 실행 방법
프로젝트 루트 디렉토리에서 아래 명령어를 실행하면 됩니다. 별도의 외부 라이브러리는 필요하지 않습니다.
```bash
python main.py
```
# 4. 기능 상세 목록
퀴즈 풀기: 저장된 문제를 순서대로 출제하고, 점수를 계산합니다.

퀴즈 등록: 문제, 선택지 4개, 정답 번호를 입력해 새 문제를 추가합니다.

목록 조회: 현재 등록된 모든 퀴즈의 제목을 번호와 함께 출력합니다.

최고 점수 확인: 역대 가장 많이 맞힌 정답 개수를 확인합니다.

데이터 유지: `state.json` 파일을 통해 문제 목록과 최고 점수를 저장합니다.

# 5. 파일 구조 및 데이터 설명
## 1. 파일 구성

main.py: 프로그램 실행 파일입니다. 메뉴 출력, 퀴즈 진행, 저장/불러오기를 담당합니다.

Quiz 클래스: 문제, 보기, 정답 데이터를 보관하고 정답 여부를 판별합니다.

QuizGame 클래스: 파일 입출력, 메뉴 루프, 점수 계산 등 전체 흐름을 관리합니다.

state.json: 퀴즈 데이터와 최고 점수가 저장되는 파일입니다.

.gitignore: `state.json`, 캐시 파일 등을 Git 추적에서 제외합니다.

README.md: 프로젝트 상세 설명 문서.

## 2.데이터 스키마 (state.json)
데이터는 아래와 같은 JSON 구조로 관리됩니다.

{
    "quizzes": [
        {
            "question": "문제 내용이 들어갑니다.",
            "choices": ["보기1", "보기2", "보기3", "보기4"],
            "answer": 1
        }
    ],
    "best_score": 5
}
state.json 읽기/쓰기 흐름
1. Load: 프로그램 시작 시 파일이 있으면 데이터를 읽고, 없으면 기본 문제 목록을 사용합니다.
2. Save: 문제 추가 또는 최고 점수 갱신 시 현재 데이터를 파일에 저장합니다.

# 6. 예외 처리 기준
숫자 입력: 공백 입력, 문자 입력, 범위를 벗어난 숫자를 다시 입력하도록 처리합니다.

비정상 종료: `Ctrl+C` 또는 `EOFError` 발생 시 가능한 경우 데이터를 저장한 뒤 종료합니다.

파일 무결성: `state.json` 파일이 없으면 기본 퀴즈로 시작하고, 파일이 손상된 경우 초기화 여부를 다시 묻습니다.
